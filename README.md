# treview[wip]
GitHub Trending viewer

# Install
```
$ go get github.com/inabajunmr/treview
```

# Usage
```
$ treview -h
Usage:
  treview is cli viewer for GitHub Trending. [flags]

Flags:
  -h, --help          help for treview
  -l, --lang string   filter by lang
  -s, --span string   trending span (default "Today")
```
